# Microball ZMK キーマップ

このリポジトリはmicroball分割キーボード用のZMKキーマップ設定です。

## キーマップエディター

以下のリンクからキーマップを視覚的に編集できます：
https://nickcoutsos.github.io/keymap-editor/

## PDFダウンロード

📄 [最新のキーマップPDFをダウンロード](../../actions/workflows/update-readme.yml) 
（GitHub ActionsのArtifactsから `keymap-pdf` をダウンロードしてください）

## 現在のキーマップ

### レイヤー一覧

| レイヤー | 説明 |
|----------|------|
| Default (Layer 0) | メイン入力レイヤー |
| Function (Layer 1) | ファンクションキー（F1-F12） |
| Number (Layer 2) | 数字・記号入力 |
| Arrow (Layer 3) | 矢印キー・ナビゲーション |
| Mouse (Layer 4) | マウス操作 |
| Scroll (Layer 5) | スクロール操作 |
| Settings (Layer 6) | Bluetooth設定・システム |

4. 変更をコミット
